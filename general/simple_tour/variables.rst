Variables
==========

Variables can be explicitly declared at any point in the program e.g 

.. code-block:: simple

	str = "A string"
	num = 123456789
	
Variable type can be hinted but this requires the hint variable to be declared before hand

.. code-block:: simple
	
	string str = "A string"
	int num = 123456789
